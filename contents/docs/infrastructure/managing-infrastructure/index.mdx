---
title: Managing Infrastructure
description: How to manage infrastructure within Lodestar Forge.
---

Infrastructure can easily be managed within Lodestar Forge, including creating and deleting them. This section covers both of these topics in detail.

## Creating Infrastructure
To create a deployment within Forge, simply follow the steps below:
1. With a project selected, select the "Deployments" tab in the sidebar.
2. Select the deployment from the deployments table.
3. Select the "Add" button from within the "Infrastructure" section.
4. Complete the form to create the infrastructure.

![Creating infrastructure](/images/create-infrastructure.png)

Once the infrastructure has been created, view the [resources](/docs/infrastructure/resources) documentation for next steps.

## Deleting A Deployment
To delete a single piece of infrastructure within Forge, simply follow the steps below.

<Note type="danger">
    Be careful! Once infrastructure is deleted, it cannot be recovered.
</Note>

1. With a project selected, select the "Deployments" tab in the sidebar.
2. Select the deployment from the deployments table.
3. Within the "Infrastructure" section, select the action menu (three dots) for the infrastructure you want to delete.
4. Select the "Delete" option from the menu.
5. Confirm the deletion by clicking the "Delete" button.

![Deleting infrastructure](/images/delete-infrastructure.png)
